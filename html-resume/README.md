## How to Run the Project Locally

You can view and review the resume by either cloning the repository or downloading it as a ZIP file.

### Option 1: Clone the Repository

1. Open a terminal or command prompt.
2. Run the following command: `git clone https://github.com/rky2208/webcohort-assignments.git`

3. Navigate to the resume project folder: `cd webcohort-assignments/html-resume`

Open the index.html file in any web browser to view the resume.

### Option 2: Download as ZIP

1. Click the Code button at the top of this repository.
2. Select Download ZIP.
3. Extract the downloaded ZIP file.
4. Open the html-resume folder.
5. Double-click index.html to view the resume in your browser.
